1. Создать папку  mkdir
2. Зайти в папку  cd
3. Посмотреть, где находишься  pwd
4. Создать файл  touch
5. Создать файл со строками  cat >> (чтобы выйти - нажать Ctrl+C)
6. Посмотреть содержимое папки в строковом виде  ls
7. Посмотреть содержимое файла  cat
8. Открыть текстовый редактор для редактирования файла  vi (режим редактирования - нажать I, сохранить редактирование - :w, выйти - :q)
9. Добавление строки в файл с сохранением предыдущего содержимого  cat >> 
10. Выйти на уровень выше  cd ..
